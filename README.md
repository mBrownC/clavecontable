# ClaveContable - Sitio Web Oficial

## Descripción

Sitio web profesional para ClaveContable, empresa de servicios contables y de recursos humanos en Chile.

## Stack Tecnológico

- HTML5 - Estructura semántica
- Bootstrap 5.3 - Framework CSS responsive
- GSAP 3.x - Animaciones profesionales
- Swiper.js - Carruseles y sliders
- JavaScript Vanilla - Funcionalidad personalizada
- Font Awesome - Iconos

## Estructura del Proyecto

```
ClaveContable/
├── index.html
├── sitemap.xml
├── robots.txt
├── README.md
├── assets/
│   ├── css/
│   │   └── custom.css
│   ├── js/
│   │   └── main.js
│   └── img/
│       ├── logo/
│       ├── favicon/
│       ├── hero/
│       └── servicios/
```

## Características

- Diseño responsive (mobile-first)
- Animaciones suaves con GSAP
- Optimizado para SEO
- Performance optimizado
- Formulario de contacto
- Integración con WhatsApp
- Google My Business ready

## Instalación

1. Clonar el repositorio
2. Abrir index.html en el navegador
3. No requiere instalación de dependencias (usa CDN)
